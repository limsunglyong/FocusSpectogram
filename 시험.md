# 지침

1. 모든 시험 내용은 반드시 앱 버전을 기록한다.
2. 모든 시험 완료된 내용은 시험-아카이브.md에 추가하고 시험.md에서는 삭제처리 한다.
3. "2. 시험 항목"은 자동 시험, 수동 시험 으로 구분하여 제시한다.
   - 자동 시험 : 스스로 할 수 있는 시험 항목을 나열하고 시험 계획을 간략히 기록한 다음, 시험 결과를 사용자에서 보여준다. 이후 다음 작업시 자동 시험 내용은 리셋하고 새로운 내용으로 변경한다.
   - 수동 시험 : 사용자가 해야하는 시험 항목을 리스트 업 한다. 섹션 "2. 시험 항목" 에서는 간략히 시험 항목 번호와 시험 내용, 판정 기준을 간략하게 표로 보여준다. 세부적인 시험 내용에 대해서는 섹션 "3. 시험 방법 및 절차"에 정리하되, 어떤 것을 시험하는 것이며 어떤 순서로 시험하는 것인지 step-by-step으로 기록한다.
4. 앱 실행에 관련된 전반적인 정보들은 섹션 "1. 시험 개요" 부분에 추가하여 작성한다.

## 3. 시험 방법 및 절차

### M6-1 ~ M6-2 : Rotation X / Zoom 슬라이더
1. `npm run dev` 후 파일을 로드해 3D 서피스를 표시한다.
2. 우상단 Viewport Settings의 Rotation X 슬라이더를 움직여 카메라 상하 각이 즉시 변하는지 확인.
3. Zoom Level 슬라이더를 움직여 확대/축소가 즉시 반영되는지 확인.

### M6-3 ~ M6-4 : Perspective 전환
1. ISO 클릭 → 등각(대각선) 시점으로 정렬되는지 확인.
2. ORTHO 클릭 → 멀고 가까운 부분의 크기 차이(원근)가 사라진 평행 투영으로 보이는지 확인.
3. 3D 클릭 → 다시 원근 시점으로 돌아오는지 확인.

### M6-5 : 양방향 동기화
1. 마우스 드래그로 시점을 회전하고 휠로 줌한다.
2. Rotation X·Zoom 슬라이더 손잡이가 현재 시점에 맞춰 움직이는지 확인.
3. 슬라이더와 마우스를 번갈아 조작해도 값이 떨리거나 튀지 않고 안정적인지 확인.
